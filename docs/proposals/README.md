# SpaceComms Proposals

This directory contains design documents for proposed changes to the SpaceComms protocol and reference implementation.

## How to Submit a Proposal

1.  Copy the template (if available) or create a markdown file titled `YYYY-MM-DD-short-title.md`.
2.  Include:
    - **Summary**: What is changing?
    - **Motivation**: Why is this needed?
    - **Specification**: Detailed technical changes (JSON schemas, logic).
    - **Compatibility**: Impact on existing nodes.
3.  Submit a Pull Request for review.
